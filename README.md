# inventory_optimization
Inventory Optimization using Demand Forecasting, ABC Analysis, and EOQ. Analyzes historical sales data to predict demand, classify high-value products, and optimize order quantities to minimize inventory costs and support efficient stock management.

## Key Components
- Demand Forecasting (ARIMA, Exponential Smoothing)
- ABC Inventory Classification
- Economic Order Quantity (EOQ) Analysis

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Statsmodels
